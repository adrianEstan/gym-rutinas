🏋️‍♂️ Rutina de Cardio + Core

🔥 1. Calentamiento / Cardio (30 minutos)

Realiza una de estas actividades durante 30 minutos para activar el cuerpo:

  • 🚶 Andar rápido
  • 🚴 Bicicleta
  • 🏃 Correr

🚶‍♂️ Guía para andar (30 minutos)

Tiempo Ritmo estimado Velocidad aprox. Objetivo
0–5 minPaso cómodo4–5 km/hCalentamiento
5–10 minPaso activo5.5 km/hEmpieza la activación cardiovascular
10–15 minPaso rápido6–6.5 km/hRitmo ideal para quema de grasa
15–20 minPaso muy rápido6.5–7 km/hSudor y respiración agitada
20–25 minPaso rápido6–6.5 km/hMantener intensidad
25–30 minPaso relajado4.5–5 km/hEnfriamiento

🚴‍♂️ Guía para bicicleta estática (30 minutos)

Tiempo Nivel de resistencia Objetivo
0–5 min2–3Calentamiento
5–10 min4–5Ritmo moderado
10–15 min6–7Esfuerzo intenso
15–20 min4Ritmo moderado
20–25 min7–8Subida fuerte
25–30 min2–3Enfriamiento

* Si tu bici va del 1 al 20, multiplica los niveles x2

🧠 Recomendaciones

  • Multiplica niveles si tu bici tiene escala hasta 20
  • Si no tiene números:
    • Resistencia baja → pedaleo muy fácil
    • Resistencia media → se nota el esfuerzo
    • Resistencia alta → como subir una cuesta

🎯 Consejo de intensidad

  • Puedes hablar con frases cortas, no mantener conversación larga
  • ¿Muy cómodo? ➜ Sube un nivel
  • ¿No aguantas 5 min? ➜ Baja un nivel
